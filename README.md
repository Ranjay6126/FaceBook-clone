📘 Facebook Clone – MERN Stack Social Media App

This project is a work-in-progress Facebook Clone built using the MERN Stack (MongoDB, Express.js, React.js, Node.js). The goal of this application is to replicate core social media features of Facebook while focusing on clean architecture, scalable backend design, responsive UI, and real-time user interactions.

The application allows users to create accounts, authenticate securely, create posts, like and comment on posts, and interact with other users in a dynamic social environment. This project is designed not just as a UI clone, but as a full-stack implementation of a real-world social media platform, emphasizing best practices in both frontend and backend development.

🚀 Features Implemented / In Progress

🔐 User Authentication (JWT based login and registration)

👤 User Profile Management

📝 Create, Edit, and Delete Posts

❤️ Like and Unlike Posts

💬 Comment System on Posts

📰 News Feed with dynamic post loading

🔎 Explore Users and Profiles

🌐 RESTful API Integration

⚡ Real-time updates (planned with Socket.io)
